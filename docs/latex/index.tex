\begin{DoxyEnumerate}
\item Make sure cmake is installed in your system.
\item Move to the parent directory and create a build folder.
\item Inside the build folder run the command 
\begin{DoxyCode}{0}
\DoxyCodeLine{cmake\ ..}

\end{DoxyCode}
 This tells CMake to find the {\ttfamily CMake\+Lists.\+txt} file which specifies how to build the game.
\item Run the command 
\begin{DoxyCode}{0}
\DoxyCodeLine{make}

\end{DoxyCode}
 And wait for everything to get built. The executable should appear in the build folder.
